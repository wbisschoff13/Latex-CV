  \lettersection{About Me}

  I'm a software engineer who builds the foundations teams need to ship reliable software fast. With a B.Eng. in Computer and Electronic Engineering, I specialize in clean system architecture, testing infrastructure, and development tooling that lets teams move confidently.

  My experience spans embedded systems (C++, RTOS, BLE), full-stack web development (ERPNext, Expo), and DevOps infrastructure (AWS, Pulumi, Docker, CI/CD). I focus on code that's easy to debug, test, and maintain.

  \lettersection{Why Me?}

  \textbf{Clean architecture that scales:} At Ingenics Digital, I designed an event-driven finite state machine for an embedded system using C++ and FreeRTOS. This created a maintainable architecture that reduced bugs and accelerated development. I don't just write code—I design systems that last.

  \textbf{AI-assisted development:} I've implemented agentic workflows using Claude Code for debugging, code review, and refactoring. Through prompt engineering and AI pair programming, I've significantly improved debugging speed and code review throughput while maintaining code quality.

  \textbf{Full-stack + infrastructure:} I've extended ERPNext workflows, built mobile features in Expo with NFC integration, provisioned AWS infrastructure with Pulumi, and deployed services including Inngest and PayloadCMS. I understand the full stack from firmware to cloud.

  I'd welcome the opportunity to discuss how my experience building clean, testable systems can help your team ship reliable software.

\end{cvletter}
